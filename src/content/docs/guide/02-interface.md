## หน้าจอหลัก (Main Window)

เมื่อเปิดโปรแกรม ETABS ขึ้นมา จะพบกับหน้าจอหลักที่ประกอบด้วยส่วนต่างๆ ดังนี้:

### 1. Title Bar

แสดงชื่อไฟล์ที่กำลังเปิดอยู่ และเวอร์ชันของโปรแกรม

### 2. Menu Bar

แถบเมนูคำสั่งหลักด้านบนสุด เป็นศูนย์รวมคำสั่งทั้งหมดของโปรแกรม:

- **File:** จัดการไฟล์, Import/Export, และการตั้งค่าพื้นฐาน
- **Edit:** แก้ไขโมเดล, Copy/Paste, Replicate (ทำซ้ำ), และจัดการ Grid/Story
- **View:** ควบคุมการแสดงผล, ซูม, หมุนภาพ, และการตั้งค่า 3D View
- **Define:** กำหนดค่าคุณสมบัติต่างๆ เช่น วัสดุ (Material), หน้าตัด (Section), และแรงกระทำ (Load Patterns)
- **Draw:** เครื่องมือวาดชิ้นส่วนโครงสร้างทั้งหมด (เสา, คาน, พื้น, ผนัง)
- **Select:** เครื่องมือเลือกวัตถุแบบเงื่อนไข (เช่น เลือกตามหน้าตัด, เลือกตามชั้น)
- **Assign:** สั่งการให้วัตถุที่เลือกมีคุณสมบัติอะไร (เช่น ใส่แรง, ใส่จุดรองรับ, กำหนด Diaphragm)
- **Analyze:** ตรวจสอบโมเดลและสั่งคำนวณ (Run Analysis)
- **Display:** ดูผลลัพธ์การวิเคราะห์ (Moment, Shear, Deformed Shape)
- **Design:** สั่งออกแบบคอนกรีต/เหล็ก และดูผลการตรวจสอบ (D/C Ratio)
- **Detailing:** สร้างแบบรายละเอียดและแปลนโครงสร้าง
- **Options:** ตั้งค่าหน่วย (Units), สี, และ Graphics Mode

### 3. Toolbars

แถบเครื่องมือทางลัดที่ดึงคำสั่งจาก Menu Bar มาไว้ให้กดง่ายๆ สามารถปรับแต่งได้โดยคลิกขวาที่บริเวณว่างของแถบเครื่องมือ

### 4. Model Explorer (สำคัญมาก)

หน้าต่างจัดการโมเดลแบบ Tree View ด้านซ้ายมือ (คล้าย File Explorer) แบ่งเป็น 4 แท็บ:

- **Model:** แสดงโครงสร้างข้อมูลทั้งหมด (Definitions, Groups, Loads)
- **Display:** ควบคุมการแสดงผลและผลลัพธ์การวิเคราะห์
- **Tables:** เข้าถึงตารางข้อมูลและผลลัพธ์ทั้งหมด (สามารถ Export ไป Excel ได้จากที่นี่)
- **Reports:** จัดการรายงานการคำนวณ

### 5. View Windows

พื้นที่ทำงานหลักสำหรับดูและแก้ไขโมเดล ปกติจะแสดง 2 หน้าต่างคู่กัน (Plan View และ 3D View)

> **Tip:** สามารถเพิ่มหน้าต่างได้สูงสุด 4 หน้าต่าง โดยไปที่เมนู _Options > Windows > ..._

### 6. Status Bar

แถบสถานะด้านล่างสุด แสดงข้อมูลพิกัด (Global Coordinates), หน่วยที่ใช้งานอยู่ (Units), และสถานะการเลือกวัตถุ (เช่น _2 Frames selected_)

## การควบคุมมุมมอง (View Controls)

การทำงานใน 3D ต้องใช้การควบคุมมุมมองที่คล่องแคล่ว:

- **Zoom:** หมุนลูกกลิ้งเมาส์ (Scroll Wheel)
- **Pan:** กดลูกกลิ้งเมาส์ค้างไว้ แล้วลาก
- **Rotate (3D Orbit):** กดปุ่ม **Shift** ค้างไว้ + กดลูกกลิ้งเมาส์ค้างไว้ แล้วลาก
- **Restore Previous View:** กดปุ่ม `F3` เพื่อย้อนกลับมุมมองก่อนหน้า

ในบทถัดไป เราจะเริ่มลงมือสร้างโมเดลจริง โดยเริ่มจากการตั้งค่า Grid และ Story กันครับ
