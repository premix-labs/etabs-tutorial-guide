## ETABS คืออะไร?

**ETABS** (Extended Three Dimensional Analysis of Building Systems) คือซอฟต์แวร์วิศวกรรมโครงสร้างระดับโลกที่พัฒนาโดย CSI (Computers and Structures, Inc.) ซึ่งถูกออกแบบมาโดยเฉพาะสำหรับการวิเคราะห์และออกแบบอาคาร (Building Systems) ตั้งแต่อาคารพาณิชย์ทั่วไป จนถึงตึกระฟ้าที่มีความซับซ้อน

### ทำไมต้องใช้ ETABS สำหรับตึกสูง?

ETABS มีจุดเด่นที่ทำให้เหนือกว่าโปรแกรมวิเคราะห์โครงสร้างทั่วไปเมื่อต้องทำงานกับอาคารสูง:

1.  **Specialized for Buildings:** มีเครื่องมือเฉพาะสำหรับอาคาร เช่น การกำหนดชั้น (Story), การสร้างผนังลิฟต์ (Core Walls), และการลดน้ำหนักจร (Live Load Reduction)
2.  **High-Rise Capabilities:** รองรับการวิเคราะห์ผลของ P-Delta, การหาคาบการสั่น (Modal Analysis), และการวิเคราะห์แรงลม/แผ่นดินไหวขั้นสูง (Response Spectrum / Time History)
3.  **Comprehensive Design:** สามารถออกแบบได้ทั้งเสา, คาน, พื้น (Composite/Concrete), และผนังรับแรงเฉือน (Shear Wall) ในโปรแกรมเดียว
4.  **Integration:** ทำงานร่วมกับโปรแกรมอื่นๆ ในตระกูล CSI ได้ดี เช่น SAFE (พื้น/ฐานราก) และ SAP2000

## ความต้องการของระบบ (System Requirements)

เพื่อให้การทำงานกับโมเดลตึกสูงลื่นไหลและประมวลผลได้รวดเร็ว แนะนำให้ใช้เครื่องที่มีสเปคดังนี้:

| Hardware     | ขั้นต่ำ (Minimum)      | แนะนำ (Recommended)                             |
| :----------- | :--------------------- | :---------------------------------------------- |
| **OS**       | Windows 10/11 (64-bit) | Windows 10/11 (64-bit)                          |
| **CPU**      | Intel i5 / AMD Ryzen 5 | Intel i7/i9 หรือ AMD Ryzen 7/9                  |
| **RAM**      | 8 GB                   | **16 GB หรือ 32 GB** (สำคัญมากสำหรับตึกสูง)     |
| **Storage**  | 6 GB (HDD)             | 500 GB SSD (NVMe เพื่อการ Save/Load ที่รวดเร็ว) |
| **Graphics** | รองรับ DirectX 11      | NVIDIA GeForce (4GB VRAM ขึ้นไป)                |

> **Tech Tip:** หากต้อง Run Analysis ตึกสูงที่มีโหมดการสั่นจำนวนมาก (Modal Cases) แนะนำให้เน้น CPU ที่มี Clock Speed สูง และ RAM จำนวนมาก

## ขั้นตอนการติดตั้ง (Installation)

1.  **ดาวน์โหลดตัวติดตั้ง:** จากเว็บไซต์ทางการของ CSI หรือตัวแทนจำหน่าย
2.  **รันไฟล์ Setup:** ดับเบิลคลิกที่ไฟล์ `.exe` และปฏิบัติตามขั้นตอน
3.  **เลือก License:** เลือกระหว่าง Standalone (เครื่องเดียว) หรือ Network (ใช้งานร่วมกันในองค์กร)
4.  **Activation:** ใส่ Activation Key ที่ได้รับหลังจากการซื้อ
5.  **Finish:** เมื่อติดตั้งเสร็จสิ้น ไอคอน ETABS จะปรากฏบนหน้าจอ

## เตรียมความพร้อมก่อนเริ่มใช้งาน

ก่อนจะเริ่มสร้างโมเดลในบทถัดไป ขอให้แน่ใจว่าคุณได้เตรียมข้อมูลเหล่านี้ไว้แล้ว:

- **แบบสถาปัตยกรรม (Architectural Drawings):** แปลนพื้น, รูปด้าน, รูปตัด
- **ข้อกำหนดการออกแบบ (Design Criteria):** มาตรฐานที่จะใช้ (เช่น ACI 318-19, มยผ.), กำลังวัสดุ (fc', fy)
- **ข้อมูลดินและแรงลม:** สำหรับกำหนดแรงแผ่นดินไหวและแรงลมตามสถานที่ก่อสร้าง

ในบทต่อไป เราจะมาทำความรู้จักกับหน้าตาและเครื่องมือต่างๆ ของ ETABS กันครับ
